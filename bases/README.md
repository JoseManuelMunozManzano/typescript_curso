# Bases de TypeScript

Proyecto creado con

```
npm init
```

Instalación de Typescript

```
npm i -D typescript
```

Generación de fichero tsconfig.json

```
npx tsc --init
```

Se crea en package.json el script build con watch mode

```
"build": "tsc -w"
```

Transpilar con

```
npm run build
```
